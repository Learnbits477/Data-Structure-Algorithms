## 🎯 About

This repository is a **structured log of daily problem-solving** across two of the most respected competitive programming platforms. Each problem is tackled with:

- ✅ A clean, well-commented **C++ solution**
- 📖 A detailed **approach explanation** with visual Mermaid diagrams
- 🧪 A **driver file** with custom test cases
- 📋 A **problem statement** documenting constraints and source link

> The goal is consistency — one problem per day, every day, with a focus on understanding patterns rather than memorizing solutions.

| File | Role |
|------|------|

| `Main.cpp` | Test driver with sample inputs and expected outputs |

## 🚀 How to Run

**Prerequisites:** GCC/G++ with C++17 support

```bash
# 1. Clone the repository
git clone https://github.com/Learnbits477/Data-Structure-Algorithms.git
cd Data-Structure-Algorithms

# 2. Navigate to any problem
cd GeeksForGeeks/94_Day

# 3. Compile
g++ -std=c++17 -O2 Main.cpp -o solution

# 4. Run
./solution
```

> **Windows users:** Use `solution.exe` instead of `./solution`, or run directly via VS Code / Dev-C++.
