## 🧩 Problem Description

Given an array `arr[]` consisting of distinct integers, check if the given array can represent preorder traversal of a Binary Search Tree (BST).

---

## 📌 Examples

### Example 1
**Input:** 
```text
arr[] = [2, 4, 3]
```
**Output:** 
```text
true
```
**Explanation:** 
Given `arr[]` can represent preorder traversal of the following BST:
```text
    2
     \
      4
     /
    3
```

### Example 2
**Input:** 
```text
arr[] = [2, 4, 1]
```
**Output:** 
```text
false
```
**Explanation:** 
Given `arr[]` cannot represent preorder traversal of a BST.

---

## 📐 Constraints

- $1 \le arr.size() \le 10^5$
- $0 \le arr[i] \le 10^5$
