# [Check if a number is power of another number](https://www.geeksforgeeks.org/problems/check-if-a-number-is-power-of-another-number5442/1)

**Problem Link:** [GeeksforGeeks - Check for Power](https://www.geeksforgeeks.org/problems/check-if-a-number-is-power-of-another-number5442/1)

Given two positive integers `x` and `y`, determine if `y` is a power of `x`. If `y` is a power of `x`, return **true**. Otherwise, return **false**.

---

## 📝 Examples

### **Example 1**
**Input:** 
```text
x = 2, y = 8
```
**Output:** 
```text
true
```
**Explanation:** 
$2^3 = 8$.

### **Example 2**
**Input:** 
```text
x = 1, y = 8
```
**Output:** 
```text
false
```
**Explanation:** 
Any power of 1 is just 1, which is not equal to 8.

### **Example 3**
**Input:** 
```text
x = 46, y = 205962976
```
**Output:** 
```text
true
```
**Explanation:** 
$46^5 = 205962976$.

### **Example 4**
**Input:** 
```text
x = 50, y = 312500000
```
**Output:** 
```text
true
```
**Explanation:** 
$50^5 = 312500000$.

---

## ⚙️ Constraints

*   $1 \le x \le 10^3$
*   $1 \le y \le 10^9$
