## 🧩 Problem Description

Given two integers $a$ and $b$ in the form of strings. Return the last digit of $a^b$.

---

## 📌 Examples

### Example 1
```text
Input: a = "3", b = "10"
Output: 9
Explanation: 3^10 = 59049. Last digit is 9.
```

---

### Example 2
```text
Input: a = "6", b = "2"
Output: 6
Explanation: 6^2 = 36. Last digit is 6.
```

---

## 📐 Constraints

- $$1 \le a.\text{size}(), b.\text{size}() \le 1000$$
- $a$ and $b$ consist only of numeric digits (`'0'` - `'9'`)
- $a$ and $b$ do not contain any leading zeros, except when the number itself is `"0"`

---

## ⏱️ Expected Complexities

| Complexity | Bound |
| :---: | :---: |
| 🕐 Time | $$O(|b|)$$ |
| 💾 Auxiliary Space | $$O(1)$$ |
