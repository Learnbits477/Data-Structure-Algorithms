## 🧩 Problem Description

Given an array `arr[]` and positive integer `k`, count total number of pairs in the array whose sum is divisible by `k`.

---

## 📌 Examples

### Example 1
**Input:**
```text
arr[] = [2, 2, 1, 7, 5, 3], k = 4
```
**Output:**
```text
5
```
**Explanation:**
There are five pairs possible whose sum is divisible by '4' i.e., (2, 2), (1, 7), (7, 5), (1, 3) and (5, 3).

### Example 2
**Input:**
```text
arr[] = [5, 9, 36, 74, 52, 31, 42], k = 3
```
**Output:**
```text
7
```
**Explanation:**
There are seven pairs whose sum is divisible by 3, i.e, (9, 36), (9, 42), (74, 52), (36, 42), (74, 31), (31, 5) and (5, 52).

---

## 📐 Constraints

- $$1 \le |arr| \le 5 \times 10^4$$
- $$1 \le arr[i] \le 10^6$$
- $$1 \le k \le 5 \times 10^4$$
