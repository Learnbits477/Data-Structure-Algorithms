## 🧩 Problem Description

Given a string `s` consisting of lowercase English letters, find the maximum number of characters between any two identical characters. If no character repeats, return `-1`.

---

## 📌 Examples

### Example 1
**Input:**
```text
s = "socks"
```
**Output:**
```text
3
```
**Explanation:**
There are 3 characters between the two occurrences of `'s'`.

### Example 2
**Input:**
```text
s = "for"
```
**Output:**
```text
-1
```
**Explanation:**
No repeating character present.

---

## 📐 Constraints

- $$1 \le |s| \le 10^5$$
- The string `s` consists only of lowercase English letters.

---
