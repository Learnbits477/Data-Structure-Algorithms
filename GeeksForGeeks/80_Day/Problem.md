## 📖 Problem Statement

Given a string `s`, determine whether its characters can be **rearranged** to form a **palindrome**.  
Return `true` if it is possible to rearrange the string into a palindrome; otherwise return `false`.

---

## 🧩 Examples

### Example 1
```
Input:  s = "baba"
Output: true
Reason: Can be rearranged to form the palindrome "abba"
```

### Example 2
```
Input:  s = "geeksogeeks"
Output: true
Reason: Characters can be rearranged to form "geeksoskeeg"
```

### Example 3
```
Input:  s = "geeksforgeeks"
Output: false
Reason: The given string cannot be converted into a palindrome.
```
